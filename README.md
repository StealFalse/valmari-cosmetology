ValMari — Учебный центр эстетической косметологии

Верстка сайта по макету Figma

О проекте

Этот сайт был создан в качестве демонстрации моих навыков HTML и CSS. Я сверстал макет учебного центра эстетической косметологии «ValMari», стремясь добиться максимального визуального совпадения с оригинальным дизайном в Figma.

Цель

Показать умение:
- Чисто и аккуратно верстать макеты
- Работать с Flexbox и Grid
- Создавать адаптивный дизайн
- Использовать CSS-анимации и hover-эффекты
- Соблюдать семантическую структуру HTML

Использованные технологии

- HTML5 — семантическая разметка
- CSS3 — стилизация, Flexbox, Grid, медиа-запросы
- Font Awesome — иконки
- Google Fonts — шрифт Montserrat
